# Qinghai-Tibet Plateau Biodiversity Habitat Risk Real-time Monitoring System

A desktop application for monitoring biodiversity habitat risks in the Qinghai-Tibet Plateau region.

## Download

Due to the large file size (~2.6GB), the compiled executable is hosted on Baidu Netdisk.

**Download Link**: https://pan.baidu.com/s/1wJNDkJdrIL5fvvc3vpr2Xw  
**Extraction Code**: tkst

## System Requirements

- Windows 10/11 (64-bit)
- No additional software required (standalone executable)

## Installation

1. Download the `.rar` file from the Baidu Netdisk link above
2. Extract the file using WinRAR or 7-Zip
3. Run `TP_HabitatRisk.exe`

## Features

- Real-time monitoring of habitat risks in the Qinghai-Tibet Plateau
- Biodiversity data visualization
- Species distribution analysis
- Habitat fragmentation assessment

## Technical Stack

- Python
- PyQt5
- GIS (Geographic Information System)
